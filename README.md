# thomasdeschamps — portfolio

Personal site for Thomas Deschamps — freelance AI & product manager.
Warm, minimal, type-led, with a few tiny things to find.

Built by hand with **Astro + Tailwind**. A little WebGL where it earns its place.

> Tip: hold the 🍅. And the Konami code does something.

## Develop

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # static output in dist/
npm run preview  # serve the build
```

## Stack

- **Astro** (static, near-zero JS)
- **Tailwind v4** (via the Vite plugin)
- Self-hosted fonts: Instrument Sans · Inter · JetBrains Mono · VT323
- **three.js** loaded on demand (only for the easter egg — never on first paint)
